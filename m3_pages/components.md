# Components — Material Design 3

Source: https://m3.material.io/components

Components are interactive building blocks for creating a user interface. They can be organized into categories based on their purpose: Action, containment, communication, navigation, selection, and text input.

## Buttons

- [Button groups](./components/button-groups) — Button groups organize buttons and add interactions between them
- [Buttons](./components/buttons) — Buttons prompt most actions in a UI
- [Extended FABs](./components/extended-fab) — Extended floating action buttons (extended FABs) help people take primary actions
- [FAB menu](./components/fab-menu) — The floating action button (FAB) menu opens from a FAB to display multiple related actions
- [Floating action buttons (FABs)](./components/floating-action-button) — Floating action buttons (FABs) help people take primary actions
- [Icon buttons](./components/icon-buttons) — Icon buttons help people take actions with a single tap
- [Segmented buttons](./components/segmented-buttons) — Segmented buttons help people select options, switch views, or sort elements
- [Split buttons](./components/split-button) — Split buttons open a menu to give people more options related to an action

## Date & time pickers

- [Date pickers](./components/date-pickers) — Date pickers let people select a date, or a range of dates
- [Time pickers](./components/time-pickers) — Time pickers help people select and set a specific time

## Loading & progress

- [Loading indicator](./components/loading-indicator) — Loading indicators show the progress of a process for a short wait time
- [Progress indicators](./components/progress-indicators) — Progress indicators show the status of a process in real time

## Navigation

- [Navigation bar](./components/navigation-bar) — Navigation bars let people switch between UI views on smaller devices
- [Navigation drawer](./components/navigation-drawer) — Navigation drawers let people switch between UI views on larger devices
- [Navigation rail](./components/navigation-rail) — Navigation rails let people switch between UI views on mid-sized devices

## Sheets

- [Bottom sheets](./components/bottom-sheets) — Bottom sheets show secondary content anchored to the bottom of the screen
- [Side sheets](./components/side-sheets) — Side sheets show secondary content anchored to the side of the screen

## All other components

- [App bars](./components/app-bars) — App bars are placed at the top of the screen to help people navigate through a product
- [Badges](./components/badges) — Badges show notifications, counts, or status information on navigation items and icons
- [Cards](./components/cards) — Cards display content and actions about a single subject
- [Carousel](./components/carousel) — Carousels show a collection of items that can be scrolled on and off the screen
- [Checkbox](./components/checkbox) — Checkboxes let users select one or more items from a list, or turn an item on or off
- [Chips](./components/chips) — Chips help people enter information, make selections, filter content, or trigger actions
- [Dialogs](./components/dialogs) — Dialogs provide important prompts in a user flow
- [Divider](./components/divider) — Dividers are thin lines that group content in lists or other containers
- [Lists](./components/lists) — Lists are continuous, vertical indexes of text and images
- [Menus](./components/menus) — Menus display a list of choices on a temporary surface
- [Radio button](./components/radio-button) — Radio buttons let people select one option from a set of options
- [Search](./components/search) — Search lets people enter a keyword or phrase to get relevant information
- [Sliders](./components/sliders) — Sliders allow users to make selections from a range of values
- [Snackbar](./components/snackbar) — Snackbars show short updates about app processes at the bottom of the screen
- [Switch](./components/switch) — Switches toggle the selection of an item on and off
- [Tabs](./components/tabs) — Tabs organize content across different screens and views
- [Text fields](./components/text-fields) — Text fields let users enter text into a UI
- [Toolbars](./components/toolbars) — Toolbars display frequently used actions relevant to the current page
- [Tooltips](./components/tooltips) — Tooltips display brief labels or messages
