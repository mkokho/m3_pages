# Material Design 3 for Flutter

Source: https://m3.material.io/develop/flutter

A framework for building beautiful, natively compiled applications from a single codebase. Support is available for Material Design 3.

## Material 3 Widget catalog

Flutter supports original M3 components. M3 Expressive is not available on Flutter.

[Material Flutter widget catalogThe Flutter team is currently working on Material Design 3 support for Material Components.](https://docs.flutter.dev/development/ui/widgets/material)
